## Python Port Scanner
Scan single host with UDP scan and OS Detection options
## Prerequisites
* ```Python3``` this code written in python3
* ```scapy``` this code required scapy package
```python
pip install scapy
```
## Usage
``` python main.py -i <target ip> [OPTIONS] ```
## Options
* ```-sU``` for UDP Scanning
* ```-OS``` for OS Detection
